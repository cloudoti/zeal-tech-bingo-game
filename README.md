# Bingo Game

Node.js Reto.

## Instalación

Usa el administrador de paquetes [npm](https://www.npmjs.com/package/page) para instalar las dependencias.

```bash
npm install
```

## Descripción

Para el desarrollo del reto se uso los siguiente:

* TypeScript
* Mocha
* Chai
* Patron de diseño Strategy
* Collecciones

```bash
npm run ts-test
```
